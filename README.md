# 3, 2, 1 – Programming Roadmap 🚀

This repository documents my personal programming roadmap as I develop my skills in computer science and software development.

The roadmap is divided into **three phases**, each focused on a different stage of learning and project complexity. The goal is to build projects consistently, document progress, and evolve toward fullstack development.

---

# 📍 The Big Picture

This roadmap follows a structured progression:

### 1️⃣ Establishing — Discipline and Fundamentals
Focus on strengthening core programming skills through structured and incremental projects.

Main activities:
- CRUD systems in C
- Landing pages (HTML & CSS)
- Competitive programming practice
- Data structures implementation in C
- Fullstack fundamentals (FreeCodeCamp)

---

### 2️⃣ Growing — Integration and Complexity
Combine technologies and tackle more complex problems.

Main activities:
- CRUD systems in C++ and Java
- Web applications with JavaScript (adding logic to the frontend)
- Python projects (automation, web crawlers, scripts)
- Continued competitive programming practice
- Data structures applied to real systems

---

### 3️⃣ Creating — Complete Real-World Systems
Build complete applications that simulate real-world scenarios.

> The goal is to finish this phase with at least three fully built, deployed, and documented fullstack projects.

Examples of projects:
- E-commerce (flower shop)
- Personal portfolio website
- Personal finance system
- Reseller management system
- Grocery helper application

---

# 🔁 Methodology

Each phase follows a repeatable weekly structure:

> Build → Apply → Reflect → Repeat

- Build: Implement the core idea or structure  
- Apply: Use it in a practical system  
- Reflect: Document what was learned  
- Repeat: Improve with the next iteration
